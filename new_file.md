# 这是一个MD格式的文档
我打算把它作为学习Git版本管理的演示文档。

## 如何将文档add到`Git`缓存区中
```
	git add readme1.txt
	git status
```
## 如何将文档的修改Commit到本地仓库
```
	git commit -m "开始编写Git，创建了一个学习文档"
```
## 安装GIT GUI Client `TortoiseGit`
- 配置TortoiseGit
- 配置HBuildeX的插件

## 如何同步到`Github`远程仓库
- 申请Github账号
- 配置ssh key
- 尝试将本地仓库与远程仓库`Git同步`
